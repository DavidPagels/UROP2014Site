About

Graph Algorithm Visualization Program is a learning/research utility comprised mainly of two components, the visual representation of the user-created graph and a language (called GraphLang) created to interact with that visual representation.

GraphLang

GraphLang is a language created to interact with the visual representation of the graph constructed by the user. GraphLang is case and space sensitive language with all keywords being lowercase. Most, if not all statements begin with a keyword such as set, declare, add, etc.

Setting Variables

set <vartype> <varname> [<attribute>] <to|=> <expression>


vartype: int, string, vertex, edge

varname: user defined variable names

attribute: needed for vertices and edges to specify what attribute to change.

to and =: may be used interchangeably

expression:the expression is evaluated and assigned to the variable
